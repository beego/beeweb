an open source framework to build and develop your web applications in the Go way.

Beego is an open-source framework to quickly build and develop Go web applications. It was inspired by tornado and sinatra, but we give you true Go experiences when you are coding, not translating! RESTful, MVC, session, cache, intelligent routing, thread-safe map, hot upgrade, and all the things you're going to fall in love with.
